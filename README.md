# Student Grade Tracker (Java GUI)

## 📌 Description
A Java Swing-based GUI application that manages student grades. It allows input of student data and calculates average, highest, and lowest scores.

## 💡 Features
- Add student name and grade
- Display all students
- Calculate average score
- Find highest and lowest grade
- GUI interface using Java Swing

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- ArrayList

## ▶️ How to Run
Compile:
javac StudentGradeTrackerGUI.java

Run:
java StudentGradeTrackerGUI
